## v0.5 — Config-Based Slot Definitions

This pre-release marks the next stable development milestone of the Edge Vision Inspection System.

### Added

- Moved shelf slot coordinates into `configs/shelf_slots.yaml`
- Moved per-slot detection thresholds into `configs/shelf_slots.yaml`
- Updated `src/shelf_config.py` to load slot definitions from YAML
- Added v0.5 test notes for baseline comparison against v0.4
- Updated README to document the v0.5 development status

### Verified

- A1, A2, and A3 slot regions still render correctly
- Object presence detection still runs successfully
- FPS and resolution display still work
- Program exits cleanly using `q`

### Pre-release note

This project is still under active development before v1.0. v0.5 improves project structure and configurability, but the system still uses manually defined regions of interest and classical OpenCV-based detection.
