# v1.0.0 — Edge Vision Shelf Inventory Monitor

This is the first complete working version of the camera-only shelf inventory monitoring system.

## Included in this release

- Live RGB camera-based shelf monitoring
- YOLO26s instance segmentation model
- Book, can, and shelf_base segmentation classes
- Automatic shelf slot detection
- Clean slot polygon mapping
- Slot capacity enrollment
- Visible item counting per slot
- Stock percentage calculation
- Empty, low, partial, almost-full, and full stock labels
- Mixed-item detection
- Count smoothing to reduce flickering
- Live OpenCV status overlay
- System architecture diagram
- Demo video and project assets
- Model training results and evaluation metrics
- Organized src package structure with runnable tools scripts

## Notes

This is a working prototype, not a commercial inventory system. Stock level is estimated from visible camera detections, so fully hidden items cannot be guaranteed from a single camera view.

The goal of this release is to show a cost-effective edge-vision approach to shelf-level inventory monitoring without depth cameras, RFID tags, weight sensors, or smart shelf hardware.
